# exocog.ai

Public website for Exocognition / ExoCog.

## Live

- Canonical: https://exocog.ai
- Redirect: https://exocognition.ai → https://exocog.ai

## Stack

Static HTML for V1. Planned migration to Astro 5 + Tailwind v4 + MDX once methodology and journal pages are added.

## Deploy

Cloudflare Pages, auto-deploy on push to `main`.

## Local preview

```bash
# V1 — open directly in browser
open index.html

# Or serve over HTTP if you want correct base paths
python3 -m http.server 8080
```
